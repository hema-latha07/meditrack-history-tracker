# meditrack-history-tracker
This repository hosts Meditrack, a web-based patient medical history tracking application developed as an foundational project during the early second year of my college curriculum. The primary goal of the project was to explore core web development concepts, structure layout logic, and understand frontend-to-backend routing mechanics.

The application serves as an introductory, user-friendly portal designed around essential healthcare management features:

Medical History Tracker: A centralized dashboard workspace tailored for organizing, reviewing, and monitoring historical patient medical records.  

Doctor Appointment Booking: A dedicated appointment interface where users can browse available consultation slots and book schedules with healthcare professionals. 

Health & Medical Tips: A curated resource section providing practical wellness insights, health tips, and preventive care guidance directly to patients. 

PHP Backend Integration: Implemented structural PHP scripting to manage pages, handle routing protocols, and simulate backend workflows.  

As an early academic project, the platform utilizes a static framework layout to demonstrate how modern digital health dashboards coordinate essential patient features. It serves as a strong baseline demonstrating fundamental UI/UX design and standard client-side architecture patterns.  

Tech Stack: HTML5, CSS3, JavaScript, PHP.
